# Soul - Landing Page Oficial (soul.dev.br)

Este repositório contém os arquivos públicos da página web do aplicativo **Soul**.

O objetivo principal deste site é disponibilizar com transparência os documentos legais e de segurança obrigatórios para os usuários, incluindo:
- **Termos de Uso**
- **Política de Privacidade**

*Nota: Este repositório armazena apenas os arquivos estáticos (HTML/CSS) do site. A lógica e a estrutura do aplicativo são mantidas de forma estritamente privada.*
